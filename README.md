# role-voting-applications
React + Node.js + MongoDB + Socket.IO 角色投票应用

* 使用Node.js构建一个REST API
* 使用MongoDB保存和检索数据
* 使用Socket.IO跟踪在线的访问者
* 使用React + Flux和服务端渲染来构建单页面应用

###运行项目：

* git clone
* npm install
* bower install
	* bower之前运行 `git config --global url."https://".insteadOf git://`
* gulp
* npm run watch
* 浏览器打开http://localhost:3000/
